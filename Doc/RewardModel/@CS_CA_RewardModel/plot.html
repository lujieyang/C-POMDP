PURPOSE

Plots the CS_CA_RewardModel

SYNOPSIS

function h=plot(RM)

DESCRIPTION 

 Plots the CS_CA_RewardModel

 Plot a 3d surface representing the reward function in the continous state
 and action spaces.
 It only works for one-dimensional spaces.
